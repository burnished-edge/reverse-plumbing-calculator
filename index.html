ℹ️ How this works (Disclaimer)

This tool determines the theoretical maximum capacity of a building by treating existing plumbing fixtures as the primary constraint. Based on the inputted square footage and occupancy types, the calculator derives an occupant load and applies the fractional method to determine required fixture counts. As you adjust the distribution of area or units using the sliders, the dashboard provides a real-time comparison between what is currently provided and what the code requires. When a deficiency is flagged, the user must redistribute or reduce the values until the warnings resolve, signaling that the proposed program is supported by the existing infrastructure.

The Math Breakdown for each section provides an audit trail for these results. It exposes the step-by-step logic used to reach each fractional value—including the raw occupant load derivation and the specific table-based formulas applied—ensuring that the values shown are directly traceable to standard code requirements.

Disclaimer: These values represent theoretical maximums based on reverse fractional capacity formulas. They should always be verified against normal, forward-calculating plumbing fixture requirements and confirmed by a licensed design professional in your specific jurisdiction.
